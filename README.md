# Eslam Ahmed - AI Engineer Portfolio

A single-page interactive portfolio built with **React** and **Vite**, showcasing my work as a Senior AI Engineer specializing in Generative AI, multi-agent systems, and production ML pipelines.

## Sections

| Section | Description |
|---------|-------------|
| **Home** | Introduction, profile photo, and quick links |
| **About** | Professional summary and background |
| **Experience** | 8 roles including Bank Misr, Microsoft, Upwork (Top-Rated), Curve AI, Udacity, and more |
| **Projects** | 17 production AI projects - click any card to expand full case studies with problem, solution, architecture, tech stack, and impact metrics |
| **Skills** | 42 skills across 6 categories: LLM & Generative AI, AI Frameworks & Vector Search, ML & Deep Learning, Backend & APIs, Cloud/DevOps/MLOps, Frontend & Visualization |
| **Education** | Academic background |
| **Contact** | Links to GitHub, Upwork, LinkedIn, and email |

## Tech Stack

- **Frontend:** React 18, JSX
- **Build Tool:** Vite 6
- **Deployment:** Vercel

## Local Preview

```bash
# Clone the repo
git clone https://github.com/eslamahmed235/Eslam_web_profile.git
cd Eslam_web_profile

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open **http://localhost:5173/** in your browser.

### Production Build

```bash
npm run build
npm run preview
```
